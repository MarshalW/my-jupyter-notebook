戒台寺依山势而建，坐西朝东，占地面积约4.3公顷，由南北两路组成 [4]。南路为主要建筑所在，自东向西建有山门、钟楼、鼓楼、天王殿、伽蓝殿、祖师殿、大雄宝殿、千佛阁、观音殿和九仙殿。山门外有地藏院，坐北朝南，一进院落。千佛阁南北两侧各有一座院落，称为南宫院、北宫院，北宫院又称牡丹院，原为行宫的一部分。南宫院南侧另建有两座小院为僧房，称为上院与下院。大雄宝殿南侧建有方丈院，为一座两进院落。东西两侧分别建有东西静室 [4]。北路为戒坛院，自东向西建有明王殿、戒坛殿、罗汉堂和大悲殿。戒坛院东侧为戒台寺塔院，其台下南北并列两座砖塔。北塔为法均墓塔，为七层八角密檐式塔。南塔为法均衣钵塔，为五层八角密檐塔 [4]。
1957年10月28日，戒台寺被北京市人民政府为北京市第一批市级文物保护单位 [3]。1996年11月20日，戒台寺被中华人民共和国国务院公布为第四批全国重点文物保护单位 [2]。

隋开皇年间，慧聚寺始建。
辽咸雍五年（1069年），慧聚寺北建戒坛及戒坛殿。
元朝末年，因战乱影响，慧聚寺受到破坏 [4]。
明宣德九年（1434年），慧聚寺开始重修。
明正统五年（1440年），慧聚寺重修竣工，明英宗赐额万寿禅寺 [1]。
明嘉靖年间，万寿禅寺大修长达七年之久，寺院规模进一步扩大 [4]。
清光绪十七年（1891年），万寿禅寺罗汉堂、千佛阁及牡丹院进行修葺 [1]。
20世纪30年代后，戒台寺因战乱受到破坏 [4]。
1949年后，戒台寺停止佛事活动，由北京市园林局经营管理，并开辟为公园 [1]。
1965年，戒台寺千佛阁被拆除，木料用作修理天坛斋宫使用 [7]。
20世纪80年代后，戒台寺经过大修再次开放，并由北京雕塑工厂重塑佛像。
1998年，戒台寺进驻僧人 [1]。
戒台寺依山势而建，坐西朝东，占地面积约4.3公顷，由南北两路组成 [4]。
南路为主要建筑所在，自东向西建有山门、钟楼、鼓楼、天王殿、伽蓝殿、祖师殿、大雄宝殿、千佛阁、观音殿和九仙殿。山门外有地藏院，坐北朝南，一进院落。千佛阁南北两侧各有一座院落，称为南宫院、北宫院，北宫院又称牡丹院，原为行宫的一部分。南宫院南侧另建有两座小院为僧房，称为上院与下院。大雄宝殿南侧建有方丈院，为一座两进院落。东西两侧分别建有东西静室 [4]。
北路为戒坛院，自东向西建有明王殿、戒坛殿、罗汉堂和大悲殿。戒坛院东侧为戒台寺塔院，其台下南北并列两座砖塔。北塔为法均墓塔，为七层八角密檐式塔。南塔为法均衣钵塔，为五层八角密檐塔 [4]。
山门山门面阔三间，单檐庑殿顶，筒瓦屋面，四角挂有风铃，门额上挂“山门殿” 斗子金匾。殿内前后贯通，中间为通洞。两侧各立有一尊泥质彩绘的护法神塑像，一为密执金刚，一为那罗延金刚。两侧与院墙相连，左右各有一个旁门。山门前有石狮子一对，并立有康熙帝手书《万寿寺戒坛碑记》 [4] [5]。 | |
天王殿 | |
大雄宝殿大雄宝殿建于月台之上，面阔五间，进深三间，硬山顶，绿琉璃瓦屋面，明间悬有乾隆帝手书赢额“莲界香林”。殿内供有明代铸造的铜质三世佛，屋顶装饰有三个蟠龙藻井。大雄宝殿前南北两侧各有配殿三间，硬山顶，筒瓦屋面，北侧为伽蓝殿，南侧为祖师殿 [4]。 | |
千佛阁千佛阁是戒台寺的藏经楼，位于大雄宝殿后的高台上，原建于辽代咸雍年间，大五脊重檐庑殿式建筑，分上下两层，各有十个大佛龛，每个大佛龛里有28个小佛龛，每个小佛龛里有三尊佛，共计1680尊佛，1965年被拆除，现建筑为复建 [4] [7]。 | |
观音殿观音殿位于千佛阁后的高台上，面阔三间，硬山顶，筒瓦屋面。南侧有房三间，为清朝末年恭亲王奕忻所建书房 [4]。 | |
九仙殿九仙殿位于观音殿后 [4]。 | |
牡丹院牡丹院为北宫院，位于千佛阁北侧，坐北朝南，分内外两重院落，中间以垂花门相连。其建筑风格是北京传统的四合院与江南园林艺术的巧妙结合，既有北方四合院的古朴，又有南方园林的秀美。迎门是一座用太湖石叠砌而成的花坛影壁。院内东、南两侧有房，前有回廊相连。西侧为靠山花坛。垂花门门额上挂有清朝末年恭亲王爱新觉罗·奕䜣所题“慧聚堂”的匾额。后院东、西、北三面的房屋皆有回廊相连。院子四角，用太湖石和花木建成的花坛各自成景 [4] [8]。 | |
方丈院方丈院位于大雄宝殿南侧，为一座两进院落。东西两侧分别建有东西静室 [4]。 |
明王殿明王殿面阔三间，歇山顶，简瓦屋面，为戒坛院的山门 [4]。 | |
戒坛殿戒坛殿位于戒坛院内，大殿平面呈方形面阔、进深均为五间，重檐盝顶，筒瓦屋面，屋顶正中平台有铜质镏金宝顶，呈金刚宝座塔样式分布。戒坛位于大殿正中，由青石砌筑而成，平面呈方形，通高3.25米，分为三层。其底层高1.4米，各边长11.3米；中层高0.95米，各边长9.6米；上层高0.9米，各边长8.1米。每层台均为须弥座造型，束腰处雕有佛龛，龛内有戒神尊。戒坛共有佛龛113座，戒神113尊。大殿顶部装饰有斗八方藻井 [4]。 | |
罗汉堂罗汉堂为配殿，位于戒坛殿南北两侧，各18间，原有泥塑彩绘的五百罗汉，现已无存 [4]。 | |
大悲殿大悲殿位于戒坛殿后，面阔五间，硬山顶，筒瓦屋面 [4]。 |
法均墓塔法均墓塔为北塔，原建于辽大康元年（1075年），明正统十三年（1448年）重修为七层八角密檐式塔。塔南有辽大安七年（1091年）王鼎撰文的《法均大师遗行碑》 [4]。 |
法均衣钵塔法均衣钵塔为南塔，为五层八角密檐塔 [4]。 |
- 爱新觉罗·奕䜣
爱新觉罗·奕䜣（1833年1月11日—1898年5月29日），号乐道堂主人，清末政治家、宗室，清朝十二家铁帽子王之一。道光帝第六子，咸丰帝异母弟，生母为孝静成皇后博尔济吉特氏。从清光绪十年（1884年）开始，到戒台寺“养疾避难”，留住长达十年之久。此间，曾出资改建了自己在寺内所住的北宫院，使之成为一座“寺中花园”，因院内广种牡丹花，所以人们俗称“牡丹院” [8]。
1957年10月28日，戒台寺被北京市人民政府为北京市第一批市级文物保护单位 [3]。
1996年11月20日，戒台寺被中华人民共和国国务院公布为第四批全国重点文物保护单位 [2]。
戒台寺位于北京市门头沟区马鞍山上。
