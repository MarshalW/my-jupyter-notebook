# 主要步骤

- 注册微信小程序
- 安装 “微信开发者工具”
- 在 “微信开发者工具” 中创建原型项目
- 进行原生小程序开发

# 注册微信小程序

地址： https://mp.weixin.qq.com/

需要创建一个单独的邮箱，用于注册:

- https://qiye.aliyun.com/ 使用 postmaster 账号登录
- 创建新的邮件组，小程序01/mini_01，按照顺序排列
- 通过邮件组成员账号接收激活小程序


# 安装“微信开发者工具”

```bash
brew install --cask wechatwebdevtools
```

# 在 “微信开发者工具” 中创建原型项目

使用模版，生成最基本的框架项目。

# 进行原生小程序开发

小程序有2种开发方式：

- 基于小程序原生api和组件api实现项目，这是最简单的实现方式
        - 直接用原生api，性能最好
        - 不利于开发复杂应用
        - 不利于复用
- 使用第三方开发框架，比如 [taro](https://github.com/NervJS/taro) 等，
        - 最终也会将代码编译为小程序原生形式
        - 性能不如直接用原生api
        - 利于复用，比如编译为支付宝小程序等
        - 利于开发部署过程的自动化
